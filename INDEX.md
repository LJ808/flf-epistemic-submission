## What this vault is, and isn't

**Is:** 9 real claims, hand-ingested via the E-Prime protocol, from real published sources, across three structurally different epistemic shapes (pooled epidemiology, adversarial debate, institutional consensus FAQ). Cross-linked via genuine wiki-links reflecting actual evidential and rhetorical relationships discovered during ingestion, not assigned in advance.

**Isn't:** A complete knowledge base for any of the three cases. Each case has dozens of additional sources in the wider literature not yet ingested (noted explicitly in each SOURCES file and in [[crux-analysis-v1]]'s missing-perspectives section). This is a representative slice sized to the competition's reading budget, not a finished system.

## How to extend this vault

1. Pick an uningested source from a case's "Not yet ingested" notes (see [[cern-official-faq]] for the clearest example — LSAG 2008 report).
2. Run the same procedure: pull the verbatim quote, force the E-Prime rewrite, note what the rewrite exposes that the original hid, file the claim record with the same YAML frontmatter schema used throughout this vault.
3. Add relation links to existing claims where the new claim supports, contests, or sits as a crux candidate relative to what's already here.
4. Re-run a crux pass once 3+ new claims exist in a case, to see whether the ranking in [[crux-analysis-v1]] changes.

No step in this process requires E-Prime fluency to *extend* — only to *ingest*. A researcher unfamiliar with E-Prime can read, link, and build on existing claim files without first learning the constraint that produced them. This is the compounding/shareability property tested directly, not asserted.
